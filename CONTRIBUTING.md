# Contribuir

Gracias por querer mejorar este proyecto.

## Como colaborar

1. Haz un fork del repositorio.
2. Crea una rama para tu cambio.
3. Ejecuta el proyecto localmente.
4. Comprueba que no subes caches, entornos virtuales ni datos temporales.
5. Abre una pull request explicando que has cambiado y por que.

## Ideas utiles

- Mejorar el modelo de goles esperados.
- Separar amistosos y partidos oficiales.
- Anadir datos Elo o ranking FIFA.
- Crear graficos de probabilidades.
- Permitir elegir equipos por consola.
- Crear una app con Streamlit.
- Anadir tests automaticos.

## Aviso

Este proyecto es educativo. No es una recomendacion de apuestas ni una prediccion garantizada.
